# OBLIK — Modular Furniture Configurator & Design Portfolio

## Overview

OBLIK is a premium interactive furniture configurator and design portfolio concept focused on avant-garde modular furniture systems and architectural objects.

The project combines editorial-style visual design with advanced frontend interactivity, allowing users to dynamically build custom sofa layouts, explore tactile material libraries, and interact with modular product systems in real time.

Built as a fully responsive single-page experience using semantic HTML, TailwindCSS, and vanilla JavaScript.

---

## Core Features

### Interactive Sofa Configurator

Users can:

* Build custom modular sofa compositions
* Add and remove seating modules dynamically
* Select different premium fabric finishes
* View real-time pricing updates
* Track total width, seat count, and configuration metrics
* Reset and rebuild layouts instantly

### Dynamic Material Library

Interactive architectural materials showcase featuring:

* Real-time material detail switching
* Dynamic descriptions and sourcing origins
* Visual selection states
* Premium editorial presentation

### Product Showcase

* Sculptural furniture gallery
* Hover-driven grayscale animations
* Responsive product grid
* Luxury-inspired product presentation

### Responsive Design

* Fully optimized for desktop, tablet, and mobile devices
* Sticky navigation with backdrop blur
* Adaptive layout system using CSS Grid and Flexbox
* Smooth UI transitions and animations

### Inquiry System

* Interactive contact form
* Client-side validation
* Toast notification feedback system
* Premium studio-style UX interactions
